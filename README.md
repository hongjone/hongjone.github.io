# Print-Optimized HTML/CSS Resume Template

This repository contains a minimalist, modern, and professional resume template built using standard HTML and CSS. It is specifically designed to look great both on the web and when printed.

## Features

- **Print-Ready**: Optimized for printing on A4-sized paper (210mm x 297mm) using `@page` CSS rules.
- **Two-Column Layout**: A clear, organized structure with a main content area and a sidebar for contact information and skills.
- **Modern Design**: Uses "Open Sans" typography and FontAwesome icons for a polished look.
- **Timeline-Style Experience**: A custom-styled experience section with a vertical timeline and decorators.
- **Customizable**: Built with CSS variables for easy control over colors, widths, and spacing.

## Getting Started

To use this template for your own resume, follow these steps:

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    ```
2.  **Edit `index.html`**: Update the content with your professional experience, education, and contact details.
3.  **Edit `style.css` (Optional)**: Adjust colors, fonts, or layout by modifying the CSS variables defined in the `:root` section.

## Usage

Simply open `index.html` in any modern web browser to view your resume. To save as a PDF or print, use the browser's print function (Ctrl+P or Cmd+P) and ensure the paper size is set to "A4".
